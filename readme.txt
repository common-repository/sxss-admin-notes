== Description ==

Adds a widget with a text field to the WordPress dashboard. Admins can cahnge the background and decide who can see and edit the notes.

https://www.youtube.com/watch?v=zvALYQ7U-o4
( Video by [Plugin Tube](https://www.youtube.com/channel/UCh6Ah2xerUqmHElwQ4pdpiA) )

Deutsch:
Das Plugin f&uuml;gt dem Dashboard ein Widget hinzu, in das Administratoren Notizen einf&uuml;gen und speichern k&ouml;nnen. Au&szlig;erdem kann die Hintergrundfarbe ge&auml;ndert, sowie eingestellt werden wer das Widget sehen und bearbeiten darf. Es wird das Datum der letzten Speicherung angezeigt.

More informations on the official plugin page: [http://sxss.nw.am](http://sxss.nw.am "sxss.nw.am")

== Installation ==

Upload the sxss Admin Notes plugin to your blog, activate it, go to 'Dashboard'.
